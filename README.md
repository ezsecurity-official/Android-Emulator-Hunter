# Android-Emulator-Hunter
Detect if the application is running on an emulator

## Features
- [x] Memory Scanning for x86 Libraries
- [x] Native check for application installer package using C++ and JNI

## Supported Emulators

| Emulator    | Supported |
|-------------|:---------:|
| BlueStacks  |     ❌     |
| LDPlayer    |     ❌     |
| NoxPlayer   |     ❌     |
| MEMU Play   |     ❌     |
| Mumu Player |     ❌     |
